Edit Web Catalog Details and Content Nodes
------------------------------------------

To edit the Web Catalog details: 

#. Navigate to **Marketing > Web Catalogs** in the main menu.

#. Hower over the |IcMore| *more actions* menu to the right of the item.
   
    * To edit the Web Catalog name and description:

        #. Click the |IcEdit|.

        #. Update the **Name** and **Description** for the Web Catalog.

        #. Click **Save** on the top right of the page.

    * To edit the content nodes:

        #. Click the |IcEditContentTree| on the top right of the page.

        #. In the *General* section:

             #) a
             #) b
             #) c

        #. In the *SEO* section:

             #) a
             #) b
             #) c

        #. In the Restriction section:

             #) a
             #) b

        #. In the *Content Variants* section:

             #) a
             #) b
             #) c

   The following page opens:

   .. image:: /user_guide/img/taxes/customer_tax_codes/FOREIGNGOVERNMENTSEditAccountTaxCodesTaxes.png
      :class: with-border

#. Update the **Code** and **Description** with new information about the Web Catalog.

#. Click **Save** on the top right of the page.

The Web Catalog is updated.

See information on `linking a tax code to a customer <../taxes/link_a_tax_code_to_a_customer.html>`_ for more information.

You can edit the association of the Web Catalog with the tax when `editing the tax rule details <../taxes/tax-rules/edit>`_ (see the respective topic for more information).
