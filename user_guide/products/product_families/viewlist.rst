Viewing and filtering all Product Families
------------------------------------------

To view all Product Families, navigate to **Products > Product Families** in the main menu.

Preview:

+-----------------+-------------+
| FIELD NAME      |             |
+-----------------+-------------+
| CURRENCIES      |             |
+-----------------+-------------+
| CREATED AT      |             |
+-----------------+-------------+
| UPDATED AT      |             |
+-----------------+-------------+

**Next steps**

You can perform the following actions with every item in the Product Families list:

 * Delete

 * View

 * Edit


You can perform the following actions at the Product Families page:

 * Create Product Family
