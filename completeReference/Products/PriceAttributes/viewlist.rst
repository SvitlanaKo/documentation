Viewing and filtering all Price Attributes
------------------------------------------

To view all Price Attributes, navigate to **Products > Price Attributes** in the main menu.

Preview:

  :start-after: begin

The following information about the Price Attributes is available in the Price Attributes list:

+-----------------+-------------+
| Name            | Description |
+=================+=============+
| PRICE ATTRIBUTE |             |
+-----------------+-------------+
| FIELD NAME      |             |
+-----------------+-------------+
| CURRENCIES      |             |
+-----------------+-------------+
| CREATED AT      |             |
+-----------------+-------------+
| UPDATED AT      |             |
+-----------------+-------------+

**Next steps**

You can perform the following actions with every item in the Price Attributes list:

 * Delete

 * View

 * Edit


You can perform the following actions at the Price Attributes page:

 * Create Price Attribute
