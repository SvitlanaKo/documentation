Viewing and filtering all Users
-------------------------------

To view all Users, navigate to **System > Users** in the main menu.

**Next steps**

You can perform the following actions with every item in the Users list:

 * View

 * Edit

 * Configuration

 * Delete


You can perform the following actions at the Users page:

 * Export

 * Create User
