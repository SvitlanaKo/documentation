Viewing and filtering all Roles
-------------------------------

To view all Roles, navigate to **System > Roles** in the main menu.

**Next steps**

You can perform the following actions with every item in the Roles list:

 * Clone

 * View

 * Edit

 * Delete

 * Create Role
