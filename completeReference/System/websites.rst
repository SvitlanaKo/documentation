Viewing and filtering all Websites
----------------------------------

To view all Websites, navigate to **System > Websites** in the main menu.

Preview:

  :start-after: begin

The following information about the Websites is available in the Websites list:

+------------+-------------+
| Name       | Description |
+============+=============+
| NAME       |             |
+------------+-------------+
| DEFAULT    |             |
+------------+-------------+
| CREATED AT |             |
+------------+-------------+
| UPDATED AT |             |
+------------+-------------+

**Next steps**

You can perform the following actions with every item in the Websites list:

 * Edit

 * View

 * Configuration


You can perform the following actions at the Websites page:

 * Create Website
