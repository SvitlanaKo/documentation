Viewing details of Taxes
------------------------

To view all Taxes, navigate to **Taxes > Taxes** in the main menu, and click on the line with the necessary Taxes.

Preview:

.. image:: /completeReference/img/Taxes/Taxes/TaxesView.png
   :class: with-border

+-------------+-------------+

+-------------+-------------+
| Rate        |             |
+-------------+-------------+

 * Edit

 * Delete
