Viewing and filtering all Requests For Quote
--------------------------------------------

To view all Requests For Quote, navigate to **Sales > Requests For Quote** in the main menu.

**Next steps**

You can perform the following actions with every item in the Requests For Quote list:

 * Edit

 * Change Status

 * Resubmit

 * Cancel

 * Delete

 * View
