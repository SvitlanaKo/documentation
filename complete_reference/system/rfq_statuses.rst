Viewing and filtering all RFQ Statuses
--------------------------------------

To view all RFQ Statuses, navigate to **System > RFQ Statuses** in the main menu.

Preview:

The following information about the RFQ Statuses is available in the RFQ Statuses list:

+------------+-------------+
| Name       | Description |
+============+=============+
| LABEL      |             |
+------------+-------------+
| NAME       |             |
+------------+-------------+
| DELETED    |             |
+------------+-------------+
| SORT ORDER |             |
+------------+-------------+

**Next steps**

You can perform the following actions with every item in the RFQ Statuses list:

 * View

 * Edit

 * Create RFQ Status
