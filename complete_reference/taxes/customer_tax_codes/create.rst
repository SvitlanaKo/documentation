Create a Customer Tax Code
^^^^^^^^^^^^^^^^^^^^^^^^^^

To create a new Customer Tax Code:

#. Navigate to **Taxes > Customer Tax Codes** in the main menu.

#. Click **Create Customer Tax Codes**.
   The following page opens:

   .. image:: /complete_reference/img/taxes/customer_tax_codes/CreateAccountTaxCodeAccountTaxCodesTaxes.png
      :class: with-border

#. Fill in **Code** and **Description** with information about the Customer Tax Code you are creating.

#. Click **Save** on the top right of the page.
   

The new Customer Tax Code is created.

**Next steps**

Link the tax code to `a customer <../link_a_tax_code_to_a_customer.html>`_ and `create a tax rule <../tax_rules/create.html>`_ that associates the customer tax code with the tax rate or tax exemption (zero rate).
